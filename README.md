\# Divya Learns Data



My journey from aspiring to accomplished Data Engineer — one small step at a time.



\## What's This?



This is my personal learning repo where I document everything I'm learning on my way to becoming a strong Data Engineer. It's part textbook, part journal, part portfolio.



I'm learning around 30 minutes a day, most days. No rushing. No cramming. Just consistent progress.



\## The Plan



The full roadmap lives in \[DE\_Learning\_Curriculum.md](DE\_Learning\_Curriculum.md).



High level, here's the journey:



1\. \*\*Foundations\*\* — the concepts every data engineer should know

2\. \*\*Linux, Command Line \& Git\*\* — the tools I'll live in every day

3\. \*\*SQL\*\* — going deep on the #1 DE skill

4\. \*\*Python\*\* — the glue language of data engineering

5\. \*\*My Company Stack\*\* — Oracle, Snowflake, Hive, Sqoop, EMR, AutoSys, Jenkins, uDeploy

6\. \*\*Modern Tools\*\* — Spark, Airflow, dbt, Kafka, Databricks, Terraform

7\. \*\*MLOps\*\* — bridging data engineering and machine learning

8\. \*\*Anthropic / Claude\*\* — integrating AI into data workflows

9\. \*\*Interview \& Portfolio Prep\*\* — turning learning into a career
